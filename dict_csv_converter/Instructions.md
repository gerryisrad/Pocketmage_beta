## English Dictionary to A-Z Files for PocketMage
Convert CSV dictionary to separate A-Z text files for PocketMage dictionary folder.
### Quick Setup
1. Download dictionary CSV from  [CloudBytes-Academy/English-Dictionary-Open-Source](https://github.com/CloudBytes-Academy/English-Dictionary-Open-Source) ,(176,023 definitions)
2. Save this script as convert.py in the same folder as your dictionary.csv file: [convert_dictionary.py](link)
3. Run the script ```python convert_dictionary.py```
4. Copy files to PocketMage:
   - Check the dictionary_by_letter folder that was created
   - Copy all .txt files (A.txt through Z.txt) to your PocketMage .dict folder
